Hire Me

If you’re looking for an experienced developer with proven track of contributions and wide participation in world class projects, then you should just hire me. I’ve been working as a freelance developer for the last three years with a main focus as a Python Backend developer. My specialties are unit testing, simple and well designed APIs, thoroughly documented code, delivering quality over quantity and working explicitly...instead of implicitly.

My expertise and interests lie in projects that are related to:

  • Rest API Design
  • Backend Web Development
  • Concurrent Applications
  • Distributed Systems (like zmq)
  • Parsers, Interpreters or Compilers

I have recently started using GoLang (Go Programming Language) and Clojure as a way to complement my Python skills. I have worked with both for some of my current clients’ projects, so I can apply this knowledge to a project of yours too.
